## Tech Stack

#### Frontend

- **Framework:** React Native
- **Features:**
  - User-friendly mobile interface
  - Responsive design for both iOS and Android

#### Backend

- **Framework:** NestJS
- **Database:** PostgreSQL
- **ORM:** Prisma

## Features

- User authentication and authorization with JWT
- Friends system with requests and notifications
